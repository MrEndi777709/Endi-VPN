## 📱 Инструкция по добавлению подписки (QR-код)

Если вы ранее использовали Endi-VPN, то знаете, как это работает. Для новых пользователей:

1. Скопируйте ссылку на raw-файл `mrendi-vpn-all.txt`:  
   `https://raw.githubusercontent.com/MrEndi777709/Endi-VPN/main/mrendi-vpn-all.txt`
2. Откройте ваш VPN-клиент (**Hiddify**, Nekoray, v2rayNG, Sing-box, Karing, Streisand и др.).
3. Добавьте новую подписку, вставив скопированную ссылку.
4. Настройте автоматическое обновление (рекомендуется раз в 12 часов).

**Альтернативный способ – QR-код:**

![QR-код подписки](https://raw.githubusercontent.com/MrEndi777709/Endi-VPN/main/mrendi-vpn-qr-code-all.png)

Просто отсканируйте код в вашем клиенте – подписка добавится автоматически.

> **Примечание:** Сейчас все файлы подписок содержат только сервер-заглушку с информацией о временной недоступности проекта.
