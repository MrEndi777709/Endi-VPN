# Endi-VPN

Проект возобновлён! Актуальные публичные VPN-конфигурации, собранные и проверенные лично.

## 📋 Что внутри подписок

- **Полная подписка** – 7 серверов (4 без обхода, 3 с обходом белых списков).
- **Без обхода** – 4 сервера (Нидерланды).
- **С обходом** – 3 сервера (Россия, Европа) – для работы в условиях «белых списков».

## 📱 Инструкция по добавлению подписки

1. Скопируйте нужную ссылку (из таблицы выше).
2. Откройте ваш VPN-клиент (**Karing**, **v2rayNG**, **Streisand**, **Hiddify** и др.).
3. Добавьте новую подписку, вставив скопированную ссылку.
4. Настройте автоматическое обновление (рекомендуется раз в 12 часов).

**Альтернативный способ – QR-код для полной подписки:**

![QR-код полной подписки](https://raw.githubusercontent.com/MrEndi777709/Endi-VPN/main/mrendi-vpn-qr-code-all.png)

Просто отсканируйте код в вашем клиенте – подписка добавится автоматически.
